# GlimOko // Premium Frontend Infrastructure ⚡

This directory contains the high-fidelity marketing and simulation engine for **GlimOko**. It is designed to reflect the surgical precision and technical excellence of our governance layer.

## 🎨 Design Philosophy: "Industrial Intelligence"

The interface is built on the principle of **High-Contrast Minimalism**. We utilize sharp 1px borders, architectural white space, and deterministic motion to create a premium engineering experience.

### Key Features (v2.0):
- **Interactive Playground**: A high-fidelity, zero-backend simulator that allows users to test PR titles/descriptions against the GlimOko heuristic engine in real-time.
- **Surgical Cursor**: A `mix-blend-mode: difference` cursor that inverts colors across the page, providing high-precision tactile feedback.
- **Scroll Journey**: A spring-based scroll engine that "assembles" the page logic as the user explores the infrastructure.
- **Industrial Typography**: Built with **Geist Mono** for that technical, command-line aesthetic.

## 🛠 Tech Stack

- **React 18** (UI Core)
- **Vite** (Build Tooling)
- **Framer Motion** (Motion & Simulation Engine)
- **Lucide React** (Iconography)
- **Geist Mono** (Technical Typography)

## 🚀 Development

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Launch Dev Environment**
   ```bash
   npm run dev
   ```

3. **Build for Production (Static)**
   ```bash
   npm run build
   ```
   *Note: The playground is entirely client-side, making the frontend 100% compatible with static hosting providers like Cloudflare Pages or Vercel.*
